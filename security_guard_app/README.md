# Security Guard App

This Flutter project will provide a cross-platform (Android/iOS) app for Super Admin and user management, QR code generation, and reporting features.

## Features
- Super Admin login
- User management (create, block, auto-block)
- Email invitations for new users
- QR code generation for locations
- Access to activity and incident reports

## Getting Started
1. Ensure you have Flutter installed: https://docs.flutter.dev/get-started/install
2. Open this project in VS Code or your preferred IDE.
3. Run `flutter pub get` to fetch dependencies.
4. Use `flutter run` to launch the app on your device or emulator.
